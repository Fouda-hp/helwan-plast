## 🗺️ خارطة الطريق للاستخدام

### للمبتدئين (أول مرة تستخدم المشروع)
```
1. اقرأ QUICK_START.md (5 دقائق)
   ↓
2. شغّل quotation_generator.py (اختبار أولي)
   ↓
3. افتح example_usage.py واختر مثال
   ↓
4. عدّل على المثال حسب احتياجك
   ↓
5. ارجع إلى VARIABLES_REFERENCE.md عند الحاجة
```

### للمستخدمين المتوسطين (لديك خبرة Python)
```
1. اقرأ QUICK_START.md (3 دقائق)
   ↓
2. راجع README.md (قسم "الاستخدام المتقدم")
   ↓
3. استورد QuotationData في مشروعك
   ↓
4. استخدم VARIABLES_REFERENCE.md كمرجع سريع
```

### للمحترفين (للدمج مع Anvil أو أنظمة أخرى)
```
1. راجع README.md (قسم "التكامل مع Anvil")
   ↓
2. انسخ quotation_generator.py إلى Server Module
   ↓
3. استخدم example_usage.py (مثال 7) كقاعدة
   ↓
4. طوّر Server Callable Function مخصصة
```

## 🎯 سيناريوهات الاستخدام

### سيناريو 1: إنشاء عرض سعر واحد سريع
**الملفات المطلوبة**: `QUICK_START.md` + `quotation_generator.py`
```python
from quotation_generator import QuotationData, generate_quotations
data = QuotationData()
data.client_name_ar = "اسم العميل"
data.total_price = "2,000,000"
generate_quotations(data)
```

### سيناريو 2: إنشاء عدة عروض أسعار
**الملفات المطلوبة**: `example_usage.py` (مثال 6)
- افتح المثال السادس
- عدّل قائمة العملاء
- شغّل الكود

### سيناريو 3: التكامل مع Anvil
**الملفات المطلوبة**: `README.md` (قسم Anvil) + `example_usage.py` (مثال 7)
- اقرأ قسم Anvil في README
- انسخ الكود الرئيسي إلى Server Module
- استخدم المثال السابع كأساس للـ Server Callable

### سيناريو 4: تخصيص التصميم
**الملفات المطلوبة**: `quotation_generator.py` + `README.md` (قسم "تعديل التصميم")
- افتح الملف الرئيسي
- ابحث عن الدوال المسؤولة عن الرسم
- عدّل الألوان/الخطوط/المسافات حسب الشرح
